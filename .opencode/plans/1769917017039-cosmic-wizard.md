## Implementation Details

### Message Splitting Algorithm
```typescript
// Split by ||SPLIT|| delimiter
const messageParts = result.text.split(/\|\|SPLIT\|\|/).map(part => part.trim()).filter(part => part.length > 0);

// Send each part with delay
for (let i = 0; i < messageParts.length; i++) {
  await sendMetaMessage({
    recipientId: args.senderId,
    text: messageParts[i],
    accessToken,
  });
  
  if (i < messageParts.length - 1) {
    await new Promise(resolve => setTimeout(resolve, 700)); // 700ms delay
  }
}
```

### Database Storage
Each message part should be stored as a separate entry in the messages table, with the same metadata but different timestamps.
